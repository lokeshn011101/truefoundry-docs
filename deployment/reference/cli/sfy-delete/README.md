# sfy delete

**Usage**

`sfy delete [OPTIONS] COMMAND [ARGS]...`

**Summary**

Servicefoundry delete resource

Supported resources:
- Workspace
- Service

**Options**

| **Option** | **Description** |
| :--- | :--- |
| -h, --help | Show this message and exit. |

**Commands**

| **Command** | **Description** |
| :--- | :--- |
| service | Delete a deployed Service and its deployments |
| workspace | Delete a Workspace |
